## 前提条件
- 您需要有一个 GitHub 账号：https://github.com
- 确保电脑已安装 Git：https://git-scm.com/downloads

---

## 第一步：初始化 Git 仓库

在项目文件夹 `c:\Users\22670\Desktop\wz4` 中，按住 `Shift` 键并右击空白处，选择「在此处打开 PowerShell 窗口」或「在终端中打开」，然后依次运行以下命令：

```bash
git init
```

---

## 第二步：提交代码

```bash
git add .
git commit -m "Initial commit - My Portfolio"
```

---

## 第三步：在 GitHub 创建仓库

1. 访问：https://github.com/new
2. **仓库名称**：输入一个名字，比如 `my-portfolio`
3. **描述**（可选）：My Portfolio Website
4. **Public/Private**：选择 Public（公开）
5. **不要勾选**「Add a README file」「Add .gitignore」「Choose a license」
6. 点击 **Create repository**

---

## 第四步：推送到 GitHub

创建完成后，GitHub 会给您显示一些命令，按照显示的内容运行，类似：

```bash
git remote add origin https://github.com/您的用户名/仓库名.git
git branch -M main
git push -u origin main
```

⚠️ **注意**：把 `您的用户名/仓库名` 替换成您实际的内容！

---

## 第五步：用 Vercel 部署（最简单！）

1. **访问 Vercel**
   - 打开：https://vercel.com
   - 点击「Sign Up」用 GitHub 账号登录

2. **新建项目**
   - 登录后，点击右上角的「Add New...」→ 「Project」
   - 找到您刚推送的仓库，点击「Import」

3. **部署**
   - 保持所有默认设置不变
   - 直接点击「Deploy」按钮
   - 等待 1-2 分钟...

4. **完成！** 🎉
   - 部署成功后，您会看到一个类似 `https://项目名.vercel.app` 的链接
   - 把这个链接分享给任何人，他们都能访问您的网站了！

---

## ✨ 自动部署

以后只要您：
1. 修改了代码
2. 运行 `git add .`
3. 运行 `git commit -m "描述您的修改"`
4. 运行 `git push`

Vercel 会**自动重新部署**您的网站！

---

## 📝 常见问题

**Q: push 的时候卡住了？**
A: 现在没有大文件了，应该很快！

**Q: 想修改网站名字？**
A: 在 Vercel 项目的 Settings → General 中可以修改。

**Q: 想用自己的域名？**
A: 在 Vercel 项目的 Settings → Domains 中可以添加。
