# 🚀 GitHub + Vercel 超简单部署（不用命令行！）

## 第一步：下载并安装 GitHub Desktop
访问：https://desktop.github.com/
下载 Windows 版本，安装并打开，用您的 GitHub 账号登录

## 第二步：用 GitHub Desktop 打开项目
1. 打开 GitHub Desktop
2. 点击 `File` → `Add Local Repository...`
3. 点击 `Choose...`，选择文件夹：`C:\Users\22670\Desktop\wz4`
4. 点击 `Add Repository`
5. 如果提示 "Initialize this repository?"，点击 "Initialize Git LFS..." 或者 "Initialize Repository"

## 第三步：推送到 GitHub
1. 在 GitHub Desktop 中，点击顶部的 "Publish repository"
2. 填写：
   - Name: `daijiaxuan`
   - Description: `My Portfolio`
   - 勾选 "Keep this code private?" → **不要勾选**（要公开）
3. 点击 "Publish Repository"

## 第四步：用 Vercel 一键部署
1. 访问：https://vercel.com
2. 点击 "Sign Up"，用 GitHub 账号登录
3. 点击 "Add New..." → "Project"
4. 找到 `daijiaxuan` 仓库，点击 "Import"
5. 保持默认设置，直接点击 "Deploy"
6. 等待 1-2 分钟...
7. 完成！您会获得一个类似 `https://daijiaxuan.vercel.app` 的链接

## 完成！
把链接分享给任何人，他们都能看到您的网站了！

## 以后修改代码怎么办？
1. 修改代码后，打开 GitHub Desktop
2. 左下角填写修改说明（比如："修改导航栏"）
3. 点击 "Commit to main"
4. 点击顶部 "Push origin"
5. Vercel 会自动重新部署！
