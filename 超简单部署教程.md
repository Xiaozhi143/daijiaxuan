# 🎯 GitHub + Vercel 超简单部署教程（不用命令行！）

## ✅ 我已经帮您完成的
- ✅ Git 仓库已初始化
- ✅ 代码已提交

---

## 第一步：下载 GitHub Desktop（最简单！）

1. 访问：https://desktop.github.com/
2. 下载 Windows 版本并安装
3. 安装后打开，用 GitHub 账号登录

---

## 第二步：在 GitHub 创建仓库

1. 访问：https://github.com/new
2. **仓库名称**：输入一个名字，比如 `my-portfolio`
3. **描述**（可选）：My Portfolio Website
4. **Public/Private**：选择 Public（公开）
5. **不要勾选**「Add a README file」「Add .gitignore」「Choose a license」
6. 点击 **Create repository**

---

## 第三步：用 GitHub Desktop 上传代码

1. **打开 GitHub Desktop**
2. 点击左上角 **File** → **Add Local Repository...**
3. 点击 **Choose...**，选择您的项目文件夹：`c:\Users\22670\Desktop\wz4`
4. 点击 **Add Repository**
5. 点击顶部的 **Publish repository** 按钮
6. 填写：
   - **Name**：和 GitHub 上的仓库名一样
   - **Description**：My Portfolio
   - **Keep this code private**：不要勾选（要公开）
7. 点击 **Publish Repository**

---

## 第四步：用 Vercel 部署（最简单！）

1. **访问 Vercel**
   - 打开：https://vercel.com
   - 点击「Sign Up」用 GitHub 账号登录

2. **新建项目**
   - 登录后，点击右上角的「Add New...」→ 「Project」
   - 找到您刚上传的仓库，点击「Import」

3. **部署**
   - 保持所有默认设置不变
   - 直接点击「Deploy」按钮
   - 等待 1-2 分钟...

4. **完成！** 🎉
   - 部署成功后，您会看到一个类似 `https://项目名.vercel.app` 的链接
   - 把这个链接分享给任何人，他们都能访问您的网站了！

---

## ✨ 以后修改代码怎么办？

1. 修改代码后，打开 GitHub Desktop
2. 左下角填写修改说明（比如：「修改了导航」）
3. 点击 **Commit to main**
4. 点击顶部 **Push origin**
5. Vercel 会自动重新部署！
